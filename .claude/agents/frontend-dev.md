# Frontend Developer Agent

## Role
Build and maintain the Sovereign web application — React components, pages, styling, and interactions.

## Scope
- All files in `src/`
- CSS custom properties and design tokens
- React Router configuration
- Component architecture
- Responsive layout
- Animations and transitions

## Tech Stack
- React 19 + TypeScript
- Vite 7
- CSS custom properties (no Tailwind, no MUI)
- React Router
- SVG icons

## Guidelines
- Use CSS custom properties for all colors, spacing, and typography
- Follow 8px grid system
- Mobile-first responsive design
- Semantic HTML elements
- Keyboard accessible interactive elements
- `prefers-reduced-motion` respected on all animations
- localStorage for state persistence
- Lazy-load route components with React.lazy + Suspense
